ALL ABOUT CSS!!

  1. Inline CSS not used as we have to write style in every heading or any other tag(avoid using this)
  2. Internal CSS it helps to write the style of h1 h2 tag in single heading ( Not used that much only when a web page is small)
  3. External CSS helps to write in diffrent file without style tag as in internal CSS.. For big site and this is considered a good practice..
